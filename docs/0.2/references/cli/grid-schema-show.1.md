```
USAGE:
    grid schema show [FLAGS] <name>

FLAGS:
    -h, --help       Prints help information
    -q, --quiet      Do not display output
    -V, --version    Prints version information
    -v               Log verbosely

ARGS:
    <name>    Name of schema
```
